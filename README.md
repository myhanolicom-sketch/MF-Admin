# MF-Admin

Admin microfrontend for Angular 21 Module Federation architecture.

## Features

- Admin dashboard with metrics
- User management interface
- Module Federation remote module
- Standalone Angular components

## Development

```bash
npm install
npm start
```

Access: http://localhost:4201

## Exposed Modules

- `./AdminComponent`: Main admin component
- `./AdminRoutes`: Admin routing configuration
- `./Module`: Complete admin module

## Technologies

- Angular 21
- Module Federation
- PrimeNG UI
- Vitest for testing
